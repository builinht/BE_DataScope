# DataScope Backend (GeoInsight)

Backend API cho hệ thống DataScope / GeoInsight, sử dụng **Node.js + Express + MongoDB**  

---

## Công nghệ sử dụng

- Node.js + Express
- MongoDB (Community Server / MongoDB Atlas)
- Mongoose ODM
- Auth0 (JWT Authentication)
- OpenAQ API
- MongoDB Tools (`mongodump`, `mongorestore`, `mongoexport`, `mongoimport`)
- Bash / PowerShell Script
